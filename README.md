# URL Shortener System

## Description

The URL Shortener System is a Python and MySQL based web application developed using Flask. The project converts long URLs into short and manageable links while storing the data in a MySQL database.

The system generates unique short codes for long URLs and redirects users to the original website when the shortened URL is accessed.

This project demonstrates practical implementation of:
- Backend Development
- Database Management
- Flask Web Development
- Python-MySQL Integration

## Features

- Generate shortened URLs
- Store URLs in MySQL database
- Automatic short-code generation
- Redirect short URLs to original websites
- Simple web-based user interface

## Technologies Used

- Python
- Flask
- MySQL
- HTML
